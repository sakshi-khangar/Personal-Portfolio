# Sakshi's Portfolio

Welcome to my personal portfolio repository! This project serves as a professional showcase of my UX design work, front-end and Back-end development capabilities, and creative projects.

## 🚀 Features
* **Modern UI/UX:** Clean, intuitive layout focused on user experience.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.
* **Component-Based Architecture:** Built with React for modularity and scalability.
* **Optimized Build:** Powered by Vite and bundled efficiently with esbuild/rollup.

## 🛠️ Tech Stack
* **Frontend Framework:** React
* **Build Tool:** Vite
* **Styling & Icons:** Lucide React, CSS
* **Bundlers/Parsers:** Rollup, Esbuild
